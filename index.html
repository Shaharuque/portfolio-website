Web developers can be split into two different areas of responsibilities and skills - frontend and backend. Frontend developers focus on programming languages such as HTML, CSS and Javascript to create the visual and interactive parts of a website - essentially the parts you can see.Backend developers are responsible for creating the logic that runs on a server which typically involves communicating with a database and manipulating data, before passing it back to the browser and presenting to the user.
